AutoHotkey Tutorial

This tutorial covers the needed necessities to start automating a plethora of (Windows) actions using AutoHotkey.

Time needed to complete this tutorial: 10 minutes

Running

AutoHotkey is only supported for Windows operating systems.

To get up and running, go to the AutoHotkey website and click Download (or here) to get the latest version. Next, install the program. After all is complete, your system should be able to execute AutoHotkey scripts.

Creating

A script is made through creating a file using the .ahk extension.
