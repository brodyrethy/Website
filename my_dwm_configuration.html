Font

I use classic IBM 8x14 VGA lookalike font called BigBlue TerminalPlus. It's my favorite because of its sharpness. I dislike any font smoothing or anti-aliasing, as I feel it causes strain.

Status Bar

I use xsetroot to aggregate information into a sort of status bar. I pack all of the commands I need into a simple bash script, of which I run upon startx.

Here's my status bar:


mpd song length |mpd Artist - Track |current volume |RAM remaining |CPU temp |battery percentage |date-month |24-hour time

It's simple, but it functions fully as a status bar. Plus, it's dead easy to modify and modular. You can download it here.

Typical Operation

I only use the tiling, and monocle layouts. I type to keep a maximum of three windows max on a single tag (workspace).

Workspaces

1-2-3 - Dev

This is the main development workspace. Most of my time spend on my computer is here. Typically, vim, and two st instances. The other workspaces are used for any overflow terminals.

4 - Media

Mpd, youtube-dl viewer (if any), and newsboat goes here.

5 - Comunication

Telegram-cli, discord, neomutt, and irc.

6-7-8 - Status terminals

Any excess terminals from opened programs go here.

9 - Web browsing

Firefox windows go here. That's pretty much it.
