Moses.rat

Created: 2020-01-26 | Modified: 2020-01-26

A modified version of Moses. A tiny RAT targeting Windows systems. I decided to keep it relatively small by limiting it to four command modules. After it's executed on the victim's machine, you can interact with the instance by appending commands to a public web server using the remote control file via SSH. The website it hosts will be accessed by the virus instance, downloading three webpages containing the commands and other info needed for the commands. After the pages are accessed, the page information is quickly purged by the remote. The command is then carried out on the instances, which then becomes dormant again, waiting for the pages to change. Right now the commands are send in plaintext, but in the future I may implement some sort of symmetric encryption.

System modules:

  • Self destruct
  • Check platform
  • Get web content
  • Get current time
  • Upload file to mega
  • Get windows username
  • Download file from mega
  • Check if file exists in path
Command modules:
  • Take screenshot
  • Execute command
  • Get public IP address
  • DDoS target web server
