Moses.rat

An modified version of Moses. A tiny RAT targeting Windows systems. I decided to keep it small and limited it to only 4 command modules.

System modules:

  • Self destruct
  • Check platform
  • Get web content
  • Get current time
  • Upload file to mega
  • Get windows username
  • Download file from mega
  • Check if file exists in path
Command modules:
  • Take screenshot
  • Execute command
  • Get public IP address
  • DDoS target web server
