Lowercase Directories

Keep dirs lowercase. There are some strong reasons for this.

You don't have to press shift for every dir name. This'll reduce RSI if you are a command line wizard like myself. You always want to make sure you cut every corner you can.

Backwards compatibility. Do you use a lot of varying operating systems? If so, you want your folders to be perfectly compatible every time. This will ensure that.
