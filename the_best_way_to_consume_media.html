The Best Way to Consume Media: Newsboat

Created: 2020-12-16 | Modified: 2020-12-25

I consume all media (news, new youtube posts, twitch going live notifications, website blog posts, etc.) in a centralized environment using Newsboat. There are a few reasons why I use newsboat over other RSS readers:

  • Vim keybindings (not default (here's a link to add them))
  • Centralization: Less distractions, just news
  • Terminal based
  • Extensibility
  • Content is saved and viewable offline
  • Macros: Allows for opening selected link with {program} (allows me to watch YouTube videos in mpv)

If that sounds apealing to you, here's how you can get it:

  • Arch based: sudo pacman -S newsboat
  • Debian based: sudo apt install newsboat
To add RSS an feed, simply paste the link into the urls file located at ~/.newsboat/. Here are a few RSS feeds to get you started:
https://lukesmith.xyz/rss.xml
https://www.youtube.com/feeds/videos.xml?channel_id=UC2eYFnH61tmytImy1mTYvhA
