YouTube Quality Picker for MPV

How to install:

  1. git clone https://github.com/jgreco/mpv-youtube-quality
  2. mkdir -p ~/.config/mpv/scripts
  3. mkdir -p ~/.config/mpv/script-opts
  4. cp mpv-youtube-quality/youtube-quality.lua ~/.config/mpv/scripts/
  5. cp mpv-youtube-quality/youtube-quality.conf ~/.config/mpv/script-opts/
After installing, press Ctrl+f to bring up the quaility picker menu. Then, use the arrow keys to move to desired quality, and press Return to select.
