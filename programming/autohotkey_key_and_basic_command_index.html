AutoHotkey Key and Basic Command Index

This page is a simplified version of the offical AutoHotkey KeyList Docs page to better serve my needs.

Just a side note, if you use Linux (especially tiling window managers) but have to use Windows for whatever reason, spend some time learning AutoHotkey. It'll basically allow you to bind keys that are almost 1:1 the same as a tiling window manager's (such as WindowsKey+b will exec firefox, WindowsKey+v will exec vim, etc.).


Running

AutoHotkey is only supported on Windows operating systems.

To get up and running, go to the AutoHotkey website and click Download (or here) to get the latest version. Next, install the program. After all is complete, your system should be able to execute AutoHotkey scripts.

F1-12 Keys
$F1
$F2
$F3
...

NOTE: The less than/greater than symbol is actually an arrow which points in the direction of the key you want to bind
(example: > points to the left, < points to the right)


Typical Syntax

<#b::Run "C:\Program Files\Firefox Nightly\firefox.exe"
Pseudo-Code: Execute program found at path\to\the.exe upon LeftWin+b chord press

$F1::Send !{F4}
Pseudo-Code: Send the keyboard command Alt+F4 upon F1 key press

CapsLock::Escape
Pseudo-Code: If CapsLock is pressed, issue an Escape key press instead
