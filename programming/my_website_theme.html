My Website Theme

Created: 2021-02-24 | Modified: 2021-02-24

You may have noticed my interesting website design. This is something I did on purpose, not because I don't know how to write CSS.

I care very little about aesthetics, I try to be purely functional in everything I do concerning computing. I too enforce optimality.

I believe this website has good design, mainly because it's focused around these design choices:

  • Consistent R G B for headers (I only use h1, h2, and h3)
    • This is a sort of mnemonic which allows you to clearly see the hierarchy in case the header sizing isn't visually apparent already
  • Bold is orange
  • Blocky font
    • Comprehensible
    • Isn't anti-aliased to trash
    • Compatible with all devices
  • Back buttons on each page (aside from articles)
    • Easy navigation
  • Dark theme
