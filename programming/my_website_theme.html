rethy.xyz Website Theme

Created: 2021-02-24 | Modified: 2021-02-24

You may have noticed the unorthodox website design. It's purposeful.

I care very little about aesthetics. I try to be purely functional in everything I do concerning computing. I must enforce optimality.

I believe this website has good design, mainly because it's focused around these design choices:

  • Consistent R G B for headers (I only use h1, h2, and h3)
    • It's a sort of mnemonic that allows you to see the hierarchy in case it isn't apparent to you already
  • Blocky font (BigBlue TerminalPlus)
    • Comprehensible
      • Not anti-aliased to oblivion and back
    • Compatible with all devices
    • Easy to render
  • Back buttons on each page (aside from articles)
    • Easy navigation
  • HTML and CSS
    • no PHP
    • no javascript
    • no perl
  • Dark theme
  • Everything bold is colored orange
  • No ads
